## 项目介绍
- 本项目用于记录学习CS336课程过程的笔记和作业。
- lecture_notes文件夹记录随堂笔记，assignments文件夹记录每个作业的源码以及作业笔记Note.md。
- 使用作业代码需进入对应作业目录，如作业1，cd assignments/assignment1-basics，即可运行相关代码
